# Airport Operations: Data Cleaning & Structuring Project

This repository documents the transformation of an unstructured operational report regarding luggage claim waiting times, converting raw data into a structured database suitable for Business Intelligence analysis.

### 🛠 The Problem
The original data was presented in a "visual report" format, which made it impossible to perform statistical analysis, automatic filtering, or integrations with tools like Power BI or Python. The data was fragmented by months across different sheets and lacked a relational database structure.

### 🚀 Solutions Implemented
To standardize the information, I executed the following actions in Excel:
* **Standardization & Cleaning:** Removed duplicates and normalized date/time formats.
* **Data Integration:** Consolidated multiple monthly reports into a single master table, adding a "Month" column for temporal analysis.
* **Advanced Excel:** Used `VLOOKUP` for data integration and Data Validation to ensure information integrity.

### 📈 Business Value
Thanks to this process, the data now allows for the calculation of critical operational KPIs such as:
* Average waiting time per airline.
* Luggage handling efficiency based on origin.
* Punctuality trends per month.
